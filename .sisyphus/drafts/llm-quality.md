## Intent
feature (LLM 출력 품질 개선). 대상: STRENGTH REPORT(training-report) + OPENAI COACH(morning-coach).

## 배경 (스크린샷 2장)
- REPORT "첫 주, 랫풀다운 폭발 상승": 랫풀다운만 '상승', 나머지 5종목 전부 '신규'. e1rm/볼륨 수치. NEXT 7 DAYS 일반 조언.
- COACH "오늘도 근성장 가자": 스파르타 응원, 7/21 랫풀다운 32kg 참조.

## 기존 코드 팩트 (이전 조사)
- coach: 최근 6세션의 "최고 세트 1개"만 봄. bodyPart/streak/volume 미전달. effort low, 800 tok.
- report: buildStats 서버서 실수치(e1rm Epley, volume). 종목=이름 단위 → 이름 3회+ 쌓여야 트렌드. bodyPart 미사용.
- bodyPart 필드 존재(가슴/등/어깨/팔/허벅지/종아리/복근/허리/기타), 코치엔 미연결.

## Q&A

### Batch 1 — 핵심 불만
- 어느 게 더 불만? → **둘 다 별로**
- 가장 큰 문제? → **너무 뻔함/일반론 · '신규' 도배 · 실행 지침 약함**
- 톤(스파르타)은 문제 아님 → 톤 유지 OK, 문제는 "알맹이".

### Batch 2 — 스타일/인사이트
- 매일 코치 핵심 대화 → **몸 전체 그림(부위 밸런스·약점) + 최근 대비 피드백** (오늘 처방/목표경로는 후순위)
- 인사이트 성격 → **패턴 발굴 + 구체 처방 균형**
- 목표 → **근비대(근육량)**

### Batch 3 — 스타일 목업
- 4개 중 선택 → **A. 밸런스+처방(중간)**: 몸 전체 그림 + 최근 피드백 + 구체 한 줄. 예:
  "랫풀다운 25→32kg 상승. 최근 5회 중 등4·하체1·가슴1 — 근비대는 균형. 하체가 비었어.
   오늘 하체 가서 스쿼트 반복부터. 등은 32kg 유지 9회."

### ★ 결정적 목표 (사용자 추가 발화)
- **"난 말라서 몸을 더 '크게', 특히 두께를 두껍게, 너비도."**
- 해석: skinny → mass gain. 두께(thickness)=등·중부(로우·데드)·가슴 density, 너비(width)=어깨 측면·등 넓이(랫/V테이퍼).
- 코치 렌즈 = 단순 '근비대'가 아니라 **체형(두께+너비/ V테이퍼) 관점**. 어깨너비·등두께·랫폭을 진단·처방해야.
